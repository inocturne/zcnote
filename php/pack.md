一个字节可以保存-128到127或0到255的数字，对应ascii码，即一个字节可以保存一个字符（0~127）,php通过ord获得对应ascii码

网络字节序是大端，主机字节序是小端，判断大小端如下：
```php
function IsBigEndian()
{
    $bin = pack("L", 0x12345678);
    $hex = bin2hex($bin);
    if (ord(pack("H2", $hex)) === 0x78)
    {
        return FALSE;
    }

    return TRUE;
}

if (IsBigEndian())
{
    echo "大端序";
}
else
{
    echo "小端序";
}

```
