### 按位与或运用
- 按位或： 可以设置多个选项
- 按位与： 可以检测是否设置该选项
```c
int a = 1;
int b = 2;
int c = 4;
int d = 8;

int n = a | d;
if ((n & a) == a) {
    echo "yes";
}
if ((n & c) == c) {
    echo "no";
}
```
或者用八进制表示如下：
```c
int a = 01;
int b = 02;
int c = 04;
int d = 010;

int n = a | d;
if ((n & a) == a) {
    echo "yes";
}
if ((n & c) == c) {
    echo "no";
}
```
在C语言中，0开头的数字表示为八进制
